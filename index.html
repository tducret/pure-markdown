# This page is pure Markdown Traditional markdown-based static sites typically require pre-processing steps to convert markdown into HTML before hosting. With this solution, **you can host the raw markdown files** while the browser handles the conversion dynamically. ## Proof Right-click on this page, and click on `View page source` (or press `[Cmd+U]`). You can also render it in your terminal with [glow](https://github.com/charmbracelet/glow) : ```sh # brew install glow # or check https://github.com/charmbracelet/glow#installation curl -s https://thispage.com | glow -p ``` ## Usage 1. **Write in Markdown**: Create a page/blog post using Markdown syntax. 2. **Add the script at the top**: Just add `<script src="script.js"></script>` at the top of the page. It will take care of the conversion into HTML. 2. **Host it**: Save your file with the `.html` suffix and host it as a static file. ### Example Page ```html <script src="script.js"></script> # My Page This page is being converted from **Markdown** to **HTML** on the fly. ``` ## Capabilities ### Emphasis It supports *italic*, **bold** and ~~strikethrough~~. ### Lists 1.
